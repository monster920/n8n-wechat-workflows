# 工作流详细说明

本文档详细介绍项目中每个工作流的功能、输入输出和使用方法。

---

## 核心公众号自动化工作流 (workflows/)

### 01_内容生成

| 属性 | 值 |
|-----|-----|
| 文件 | `01_内容生成.n8n` |
| 功能 | 使用 DeepSeek AI 根据主题生成公众号文章 |
| 触发方式 | POST `/webhook/wechat-public-account-content` |
| 状态 | ✅ 活跃 |
| 依赖 | 飞书多维表格、DeepSeek API |

**主要节点**:
- Webhook 接收请求
- 获取飞书多维表格数据
- DeepSeek 生成内容
- 内容处理和存储

---

### 02_图片生成

| 属性 | 值 |
|-----|-----|
| 文件 | `02_图片生成.n8n` |
| 功能 | 使用阿里云通义万相生成封面图和配图 |
| 触发方式 | POST `/webhook/wechat-public-account-image` |
| 状态 | ✅ 活跃 |
| 依赖 | 飞书多维表格、阿里云 DashScope API |

**主要节点**:
- 提取图片提示词
- 调用通义万相 API (wan2.6-t2i)
- 图片上传至飞书
- 更新多维表格

---

### 03_排版

| 属性 | 值 |
|-----|-----|
| 文件 | `03_排版.n8n` |
| 功能 | 将生成的内容转换为微信公众号 HTML 格式 |
| 触发方式 | POST `/webhook/wechat-public-account-typesetting` |
| 状态 | ✅ 活跃 |
| 依赖 | 飞书多维表格 |

**主要节点**:
- 获取文章内容
- HTML 格式转换
- 样式美化
- 生成排版结果

---
